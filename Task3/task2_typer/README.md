# Task 2 – Typer CLI JSON Tool

## Description
A simple CLI using **Typer** to manage JSON data with two commands:

- **add NAME AGE** → Adds a JSON entry with a timestamp.  
- **last** → Shows the last 10 saved entries.

  
## How to run
```bash
pip install typer
python main.py add John 25
python main.py last
```

All data is stored in `data.json`.
